# Ecommerce Best Practices

---

## Funnel Structure

Required order:

1. Hero
2. Categories
3. Products
4. Trust strip
5. Social proof
6. FAQ
7. CTA

---

## Trust Signals

Must include:

- Secure checkout
- Delivery info
- Returns policy

Placeholders allowed.

---

## Conversion Rules

- Products must appear within first 2 scrolls
- CTA visible above fold
- Navigation sticky

---

## Performance

Avoid:
- Video hero by default
- Heavy parallax
